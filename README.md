# Bored Master — Things To Do When Bored

**[→ Find Something To Do at boredmaster.com](https://boredmaster.com/)**

43 curated anti-boredom activities with a Smart Pick algorithm, mini-games, and creative challenges. Built for when scrolling doesn't cut it.

## What's Inside

- **43 activities** organized by category: phone games, creative projects, outdoor challenges, social games, mini-skills
- **Smart Pick** — algorithm recommends the activity most likely to work for your situation based on global user ratings
- **Mini-games** playable directly in the browser: Connect 4, Tic-Tac-Toe, reaction games, drawing canvas
- **Global leaderboards** — compete on tap speed, reaction time, and other scored games
- **Solo and group** activities — filters for alone vs. with others, quick vs. extended time

## Activity Categories

- 📱 **Phone Games** — Tap Speed Test, Shake to Decide, Swipe Reaction Game, Phone Tilt Maze
- 🎨 **Creative Projects** — Soap Carver, One-Thumb Drawing, Eyes-Closed Portraits
- 🌿 **Outdoor Curiosities** — Grass Tasting, Balance Challenges, Nature Games
- 👥 **Social Games** — Reaction Speed Battle, Simultaneous Portrait Drawing, Story Chain
- 🎓 **Mini-Skills** — Card Tricks, Juggling, 6-Word Stories

## Live Site

**[boredmaster.com](https://boredmaster.com/)** — the full activity library with Smart Pick, leaderboards, and browser-playable games.

## Stack

- React 19 + Vite + TypeScript + Tailwind CSS
- Cloudflare Pages + Cloudflare D1 (SQLite) for global leaderboards
- Cloudflare Pages Functions for serverless score API
